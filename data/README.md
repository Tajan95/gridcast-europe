# Datenablage

Lokale Struktur:

- `raw/`: unveränderte OPSD-Downloads
- `external/`: optionale CORDEX-, TYNDP- oder andere Szenariodaten
- `processed/`: bereinigte, zusammengeführte Modelldaten

Große Dateien werden nicht in Git versioniert. Jeder Download und Verarbeitungsschritt soll per Notebook oder Skript reproduzierbar sein.
